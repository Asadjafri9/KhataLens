Create a pixel-faithful, production-quality React + TypeScript + Tailwind project called KhataLens.

IMPORTANT:
This is not a generic SaaS template. This must feel like a premium, bold, editorial, investor-ready hackathon product with the same artistic direction and interaction feel as the Flux-style reference UI: sharp typography, strong contrast, burgundy accents, warm off-white background, minimal radius, premium motion, and slightly experimental visual energy.

The product is:
KhataLens — an AI-powered paper-to-ledger assistant for small shopkeepers in Pakistan. It converts handwritten udhaar notebook pages into structured digital customer ledgers, highlights overdue balances, and generates ready-to-send WhatsApp payment reminders.

The final output should include:
1. A premium marketing landing page
2. A clickable product-style app prototype section / pages
3. Strong visual continuity between marketing and product UI
4. Clean reusable components and production-quality frontend structure

Tech requirements
- React
- TypeScript
- Tailwind CSS
- Component-driven architecture
- Responsive across desktop, tablet, mobile
- Use Tailwind utilities plus small custom CSS for reveal effects, scribble underline, subtle grid backgrounds, and 3D card/flip motion
- Avoid generic gradients and bloated glassmorphism
- Keep it polished, sharp, premium, and demo-friendly
